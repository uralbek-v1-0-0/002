Assalomu Alaykum!

Bugun 3-fevral Yakshanba. 2019-yilning 34-kuni

Bugun birinchi marta o‘zimning ranglar bilan ishlangan veb saytimni yaratdim.

To‘g‘ri, bu sayt dasturchilar uchun oddiydir, lekin shuuni yaratish ham men uchun ancha qiyin bo‘ldi, oxiriyam baribir uddaladim.

Haliyam kamchiliklarim ko‘p, lekin asta sekin ularniyam to‘g‘irlab olaman Xudo hohlasa.

Muximi o‘rganishdan to‘xtab qolmayman!...

O‘zim orzu qilgan o‘zimning eng chiroyli saytimni bir kuni baribir yasab chiqaman.

Ushbu veb-saytni yaratishda foydalangan usullarim...

HTML va CSS (w3schools ko‘magida...).

Oldiniga HTMLni yozib chiqish kerak.

(HTMLni yozishda men ko‘pincha < div > bilan < p > dan foydalandim.)

Keyin CSSni HTMLga ulash kerak

HTMLga CSS < link > orqali ulanadi. Masalan:

< link rel="stylesheet" href="style.css" >

Keyin CSS yozib chiqiladi.

CSSni bilganimcha yozdim. Chunki internetda yetarlicha o‘zbekcha ma‘lumot topolmadim. Topganlarim ham inglizchadan tarjima qilingan va xatolari ko‘p.

Shunday bo‘lsayam iloji boricha o‘zimdan qo‘shib bo‘lsayam to‘g‘irlab olishga majbur bo‘ldim.

Harna, yo‘qdan ko‘ra bor deydiku...

HTMLdagi yozuvga shakl va rang berish uchun < tag class="name" >Yozuv< / tag > usulidan foydalanish qulayroq ekan. Ham deganda bu CSSdayam oson usul ekan.

Tepadagi class="name"ning namesi CSSda ham oson yoziladi. Masalan 

.name {
    uslub: qiymat;
    }

Misol uchun men tepadagi ko‘k head qismni yasashda quyidagi kodlarni yozdim.

< !DOCTYPE html >
 < html >
  < head >
   < meta charset="utf-8" >
   < meta name="viewport" 
        content="width=device-width, 
        initial-scale=1.0" >
   < link rel="stylesheet" href="style.css">
  < / head >
  < body >
   < div class="yuqori" >
    < b class="logotip">Uralbek UZ< / b >
  < / div >
 < / body >
< / html >

Keyin CSSga o‘tib pastdagi kodni yozdim.

html{box-sizing:border-box}
   *,*:before,*:after{box-sizing:inherit}
html{-ms-text-size-adjust:100%;
   -webkit-text-size-adjust:100%}
body{margin:0}
.yuqori {
    top: 0;
    padding-top: 8px;
    padding-left: 18px;
    padding-bottom: 4px;
    background-color: #0095b6;
}
.logotip {
    font-size: 24px;
    color: white;
}

(bu yerda box-sizing:border-box HTML saxifani butun ekran bo‘ylab yoyadi. *after, *before esa kontentni brauzer kengligiga to‘g‘irlab joylashtiradi. -ms-text-size-adjust bilan -webkit-text-size-adjustning qanday karomat ko‘rsatishini to‘g‘risi bilmadim, ammo ularni o‘chirib tashlab ham bo‘lmas ekan. O‘chirib tashlansa oraliqda joy qoldirib ketar ekan. Shu uchun men ham buni qo‘shib qo‘yaqoldim.)

Natijada saxifaning tepadagi xira-ko‘k head ko‘rinishi hosil bo‘ldi.
